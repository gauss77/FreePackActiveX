Custom Control che permette di scrivere e leggere nei file con
estensione .INI Attraverso semplici proprietà o utilizzando
i metodi che il controllo mette a disposizione è possibile facilitare
qualunque operazione sui file INI.

Proprietà Descrizione
Filename Restituisce o imposta il nome del file INI su cui scrivere o leggere.
Section Restituisce o imposta il nome della sezione, da scrivere o leggere.
Key Restituisce o imposta il nome della chiave, da scrivere o leggere.
Value Restituisce o imposta il valore, da scrivere o leggere. Impostando il valore
della proprietà dopo aver settato le altre proprietà verrà
automaticamente scritto il valore nel file INI. Leggendo la proprietà
verrà automaticamente letto il valore dal file INI.
Default Imposta i lvalore di default da restituire attraverso la proprietà Value
se la chiave o la sezione non è presente nel file INI.

Metodo Descrizione
ReadINI(Section$, Key$, Default$, FileIni$) Legge i dati dal file INI
WriteINI(Section$, Key$, Value$, FileIni$) Scrive i dati dal file INI
