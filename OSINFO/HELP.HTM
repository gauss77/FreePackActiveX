PRIORE OS Info Control
Copyright (c) 1998/99 D.Priore

Custom Control che permette di ricavare le informazioni più importanti
sul sistema operativo in esecuzione. Attraverso semplici proprietà si
possono ricavare tutte le informazioni più essenziali del sistema
e in aggiunta due metodi per estendere le funzionalità del codice VB.

Proprietà Descrizione
ComputerName Restituisce il nome del computer in rete.
UserName Restituisce il nome dell'utente loggato nel sistema.
BuildNumber Restituisce la versione (revisione) del sistema oprativo.
MajorVersion Restituisce la versione maggiore del sistema operativo.
MinorVersion Restituisce la versione minore del sistema operativo.
DesktopHandle Restituisce l'handle del Desktop.
DesktopIcons Restituisce o imposta lo stato di visualizzazione delle icone sul Desktop.
PlatformID Restituisce il tipo di sistema operativo in esecuzione.
ProcessorType Restituisce il tipo di processore (CPU) montato nel sistema.
SoundCard Restituisce True se la scheda sonora è installata nel sistema.
StartBar Restituisce o imposta lo stato di visualizzazione della barra delle applicazioni di Windows.
SysKeysDisabled Abilita o disabilita il riconoscimento della pressione dei tasti di sistema (es. ALT+TAB, CTRL+ALT+DEL).
CreateTempFile Restituisce o imposta se il fie temporaneo deve essere automaticamente creato.
TempFile Restituisce il nome del primo file temporaneo libero disponibile.
TempFileSuffix Restituisce o imposta il suffisso di creazione del nome del file temporaneo.
TempPath Restituisce o imposta la directory per i file temporanei.
TitleColor Restituisce o imposta il colore per la barra del titolo delle finestre nel sistema.
WinPath Restituisce la directory di Windows.
WinSysPath Restituisce la directory di sistema di Windows.
WinStart Restituisce il tempo, in ticks, di esecuzione di Windows.

Metodo Descrizione
Delay(Seconds%,[Events]) Esegue una attesa (Delay) specificata in secondi (se viene specificato False nel parametro
Events il sistema si mette in attesa finche non finisce la pausa).
ShellPlus(command$) Esegue un programma e congela l'applicazione chiamante finche l'applicazione secondaria
che è in esecuzione non termina.
